Scoutomic

SERT's Scouting Application

What is Scoutomic?

Scoutomic is SERT's scouting application, which will be used at competitions to keep track of team information and help estimate the best alliance partners.

Awesome! How does it work?

Scoutomic can be used in a variety of different ways, each with its own benefits.

Digital

By having every scout at the event use their own computer with Scoutomic installed, everyone is instantly updated with the latest scouting data, and can therefore make quick scouting decisions when it comes to the all-important alliance selection.

Hybrid

When using a hybrid scouting system, some scouts use paper-and-pencil to record match information, then pass those papers off to a team that enters data into Scoutomic. This system – though not always as up-to-date as the digital system – is much cheaper and does not require all scouts to have laptops. In fact, some scouts may want to enter data using their own laptop, and, as all data is sent to a central database (see below), this would be a perfectly fine system, too.

Cool! What if I want more than one person to enter data?

Fear not! All data is sent to an online database (which can be hosted by your own team, see below), and, if you're offline, the data will be automatically sent to the servers once you reconnect. This means that all your scouts are always up to date, and can see the latest information.

Fantastic! How do I get started?

Simply download Scoutomic, start it up, and you'll be good to go!
